# LetterCounter
 Web приложение, которое вычисляет количество символов в строке

## Стек технологий:

1. Java 17
2. Maven 3.8.6
3. Spring Boot, Web 3.1.3
4. Thymeleaf
5. JUnit Jupiter

## Формат входящих и выходящих данных 

Вход: строка
Выход: строка

Входная строка может состоять только из строчных букв английского алфавита, 
в случае нарушения этого правила вы получите ошибку.

## Пример запроса

Запрос: `/main/result`

Вход: `aaaccclllll`

Выход: `l: 5, a: 3, c: 3`
