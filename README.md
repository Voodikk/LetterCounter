# LetterCounter
 Web приложение, которое вычисляет количество символов в строке

## Стек технологий:

1. Java 17
2. Maven 3.8.6
3. Spring Boot, Web 3.1.3
4. Thymeleaf
5. JUnit Jupiter

## Запуск приложения

Для начала нужно скачать файл репозитория, распаковать его в удобное для вас место. 

Далее вам понадобится среда разработки, лично я пользуюсь IntelliJ IDEA 2021.3 (Ultimate Edition). Откройте проект, используя вашу среду разработки,
убедитесь, что у вас установлена Java 17, а также Maven 3.8.6. Проверьте, правильно ли импортировались все библиотеки в файле `pom.xml`. 

Можете запускать приложение. После успешного запуска, оно будет доступно в любом браузере по адресу `localhost:8080/main`

## Формат входящих данных 

Строка может состоять только из строчных букв английского алфавита, 
в случае нарушения этого правила вы получите ошибку.
